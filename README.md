# Web Donasiku

Aplikasi donasi berbasis Laravel untuk mencatat dan mengelola donasi secara digital.

## Fitur
- Input donasi
- Riwayat donasi
- Kategori donasi
- Data donatur

---

## Tentang Laravel

Laravel adalah framework PHP dengan sintaks elegan untuk membangun aplikasi web modern.
